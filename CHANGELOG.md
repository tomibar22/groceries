# 📝 יומן שינויים (Changelog)

כל השינויים המשמעותיים בפרויקט יתועדו כאן.

## [1.0.0] - 2025-11-29

### ✨ תכונות ראשוניות

#### ליבה
- **מאגר מוצרים**: מוצרים נשמרים לצמיתות ב-database
- **רשימה פעילה**: רשימת קניות נוכחית עם מיון לפי תאריך
- **חיפוש חכם**: Autocomplete עם Fuse.js לחיפוש fuzzy
- **הוספה מהירה**: Enter להוספה מיידית
- **סנכרון Real-time**: שינויים מסתנכרנים מיידית דרך Supabase
- **ניהול כמויות**: כפתורי +/- לעריכת כמות מוצר
- **סימון נקנה**: Checkbox לסימון מוצרים שנקנו
- **מחיקה**: הסרת מוצרים מהרשימה הפעילה

#### Mobile-First
- **כפתורים גדולים**: Touch targets של 48x48px מינימום
- **Pull to Refresh**: משיכה למטה לרענון
- **Swipe to Delete**: החלקה שמאלה למחיקה
- **Haptic Feedback**: רטט קל בלחיצה על checkbox
- **Sticky Header**: שדה חיפוש נשאר למעלה תמיד
- **RTL Support**: תמיכה מלאה בעברית
- **Responsive**: עובד מעולה על כל גודלי מסך

#### PWA
- **Installable**: אפשרות להתקין כאפליקציה
- **Offline Ready**: Service Worker לעבודה offline
- **Manifest**: PWA manifest מלא
- **Icons**: תמיכה באייקונים 192x192 ו-512x512

#### UX/UI
- **עיצוב נקי**: גרדיאנט סגול-כחול עם כרטיסיות לבנות
- **אנימציות חלקות**: 60fps transitions
- **Empty State**: הודעה כשהרשימה ריקה
- **Loading States**: אינדיקטורים לטעינה
- **ניקוי מהיר**: כפתור לניקוי כל המוצרים שנקנו

### 🛠️ טכנולוגיות

- React 18.2
- Vite 5.0
- Supabase 2.39
- Fuse.js 7.0
- vite-plugin-pwa 0.17
